Sin usar expresiones regulares:

Caso 1. Usar javaScript para validar un DNI. Formato ##.###.###-A

Caso 2. Usar javaScript para validar una cuenta corriente. Formato AA## #### #### ## ########

Caso 3. Usar javaScript para validar una fecha. Formato ##/##/####

Caso 4. Usar javaScript para validar dos fechas. Formato ##/##/####. Ambas deben ser del año actual, la primera debe ser inferior a la segunda.


Caso 5. Usar javaScript para validar que una provincia está dentro del listado de provincias españolas. Se deben eliminar los espacios antes y después del texto que se introduzca. Se compara tanto en mayúsculas como minúsculas.
